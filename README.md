## PROMPT FOR THUMBNAIL GENERATION

Be sure to customize the parts in \*\***bold****:

```text
Create a 16:9 YouTube thumbnail for an editorial political news series. 
The style should be semi-illustrated, painterly, modern, serious, and professional, similar to a European political magazine. 

Use cinematic, high-contrast lighting with a bold but controlled color palette. 
The **left** side of the image should be visually calm and partially empty for headline placement.  

The composition should be clean with clear **right-to-left** visual hierarchy and sharp foreground elements.   
Add a dynamic geopolitical collage on the **right** side of the image.   
The overall style should remain semi-illustrated, painterly, editorial, and suitable for a recurring daily news series.
Avoid photorealism, cartoon style, exaggerated caricatures, memes, stickers, watermarks, or low-resolution artifacts.
The topics to depict in the thumbnail are:
- **TOPIC_1**
- **TOPIC_2**
- **TOPIC_3**
```
